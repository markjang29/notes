# 사용자 접속 환경 정본

상태: 사용자 확인 완료
최종 확인: 2026-07-15 (KST)

## 기본 접속 방식

- 사용자는 주로 집 밖에서 Codex의 원격 연결을 통해 작업을 지시한다.
- 이 연결은 집 Windows PC의 화면을 보여 주거나 조작하는 원격 데스크톱이 아니다.
- 따라서 집 PC의 Chrome에 로그인·승인·QR 화면을 열어 두고 사용자가 직접 확인하라고 요청하면 안 된다.

## 운영 원칙

- 사용자 확인이 필요한 인증은 휴대폰에서 직접 열고 처리할 수 있는 경로를 우선한다.
- 집 PC 화면 조작이 반드시 필요하면, 사용자가 실제 원격 데스크톱을 사용할 수 있는지 먼저 확인한다.
- 일회용 인증 주소는 만료 시간을 고려해 사용자가 준비된 시점에 새로 발급한다.
- 비밀번호, OTP, 쿠키, 토큰은 채팅이나 Git에 저장하지 않는다.

## 현재 적용 대상

- AWS Tailscale 등록은 집 PC Chrome 인계 방식이 아니라 휴대폰 직접 승인 방식으로 전환한다.

## 검증 기록

- 2026-07-15: AWS 서버와 Android 휴대폰을 같은 Tailscale 사설망에 등록했다.
- AWS의 설정 웹은 Tailscale Serve HTTPS를 통해 로컬 8006 서비스로 연결했다.
- 사용자가 외부의 휴대폰에서 설정 웹이 표시되는 것을 직접 확인했다.
- 기존 공개 포트는 신규 콘솔의 기능 검증과 전환 결정 전까지 유지한다.
