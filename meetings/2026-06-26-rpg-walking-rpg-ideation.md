## 1. 의제 & 흐름 (시간순, 누락 금지)

### 1.2 이사님 1차 아이디어
- 레퍼런스 **"Charlie"** 운동앱: 걷기 → 2D 도트 캐릭터 건강, 광고→코인→코스튬.
- 제안: RPG 캐릭터 육성(경험치/아이템)을 **게임 내 + 외부활동(걷기)** 로. 걷기에 **더 큰 베네핏**. 걷기 포인트로 다수 캐릭터 육성 → **유저 간 대결**.
- 요청: "팀장 상업적 의견 + 아이디어 따로 받아적어 + 토론."

### 1.3 팀장 1차 의견 (상업 관점)
- **강점:** 외부행동→게임보상 루프(리텐션 핵), 걷기=매일 자동 콘텐츠(저비용), 2D 도트+코스튬(검증 수익).
- **리스크:** 🔴 레드오션 / 🔴 코어(전술SRPG) vs 캐주얼(피트니스) 정체성 충돌 / 🟡 P2W 인식 / 광고 ARPU 한계.
- 제안: **융합형**(전술 전투 깊이 + 걷기 성장 루프).
- 산출: `ideation/05-user-idea-walking-fitness-rpg.md` (원안 보존, 팀장 메모 별도 섹션).

### 1.4 이사님 2차 의견 — 방향 구체화
- 방향선언: **"걷기 육성 게임의 틈을 노리자."**
- 메커니즘: 술래잡기/경찰-도둑(대시포인트 필요) · 현실 지도 파밍 · 영토 수호/공성.
- **원칙(명시): "걷기에 보상이 확실하면서, 페이투윈은 막아야 한다."**
- 토론 아젠다 `work-queue.md` 정식 등록.

### 1.5 1차 다각도 검증 (3 에이전트 병렬)
- **열광유저 / 상업 / 회의** 3 관점 → **만장일치 "헌법 한 줄"**: *"걷는 사람이 이긴다. 돈으로는 코스튬을 살 수 있지만 결투에서는 이길 수 없다."*
- 충돌: 상업 "틈새 B+" vs 회의 "D+9 자멸". RUMOR×걷기 영토 결합 시그니처 부상.

### 1.6 🔥 비관 집중 디깅 (이사님 지시)
- **웹 실측** + **Codex 구조·경제학 진단** 병렬.
- **핵심 판정:** "비동기 위치기반 영토 PvP"는 밸런스 문제가 아니라 **구조적 불가**.
  - P2W 막아도 **T2W**(시간 많은 자가 이김). 새벽4시 연맹 석찬은 최적해.
  - 수익 수학: DAU 50k~100k + 전환율 3% 필요. 코스튬-only 인디 85% 재정 실패.
  - 화이트스페이스 **아님**: Orna/Turf/Ingress 니치. Niantic조차 PGO 외 전부 실패.
  - 완화책(캡/스케일링/시즌리셋)은 **미끼** — 손실 이동만.
- **건설적 전환 1안(강추):** "걷기 후 전술 PvP" — 걷기=입장연료/성장/표현, PvP=위치 분리 **동기화 아레나**(공정). = 기존 전술 SRPG 정체성 부활.
- 산출: `.reviews/walking-rpg-{skeptic-web,codex}-20260626-01.md`

### 1.7 이사님 3차 의견 — 방향 정제 + 내러티브 강조
- 지역별 **PvE** 가능 / 친구랑 같이 걸어 **입장재화 2배**.
- **T2W 재정의:** "보편적 걷기 시간 채우면 게임 내 기회 손해 안 볼 정도면 충분." (전력 우위 아닌 **기회 균등의 캡**)
- **소설적 요소 = 아주 중요:** 매력적 메인스트림 시나리오, 스토리적 당위성, 캐릭터 매력·갈등.
- 부가: 유저 **남녀 분할 → 게임을 통한 로맨스 기대**.
- ⭐ 이사님의 T2W 재정의가 Codex "살 수 있는 유일한 조건"과 **정확히 합치** (독립 도달).

### 1.8 2차 디깅 (내러티브 + 소셜, 병렬)
- **정합 발견:** 메커니즘팀(소셜)과 내러티브팀이 **독립적**으로 같은 구조 도달 → 차별의 뼈대.
  - 내러티브 α **"잃어버린 길"**(길이 끊겨 정지한 세계, 걸음이 길을 잇는다). 핵심 갈등: '길 잇는 자' vs '길 끊는 자(끊어진 발자국)' — **적도 걷는다**.
  - "동행의 서약"(두 발 같은 방향=길 2배) = 같이걷기 입장권 2배 메커닉과 정합.
- **T2W 3겹 캡:** ① 입장권 로그캡 ② 하루 N판 상한 ③ **걷기→전투력 0기여(핵심)**. Codex 사망조건 7개 중 6개 절단.
- **로맨스:** 유저×유저 위치 로맨스(데이팅)는 인디 안전비용 폭증=불가 → **캐릭터 간 내러티브 로맨스(A안)**.
- **남녀:** 성별 고정 ❌ → **진영·내러티브 분기**(성별 무관, 크로스 진영 매칭).
- 산출: `.reviews/walking-rpg-{narrative,social}-20260626-01.md`

## 2. 핵심 결정 (확정)

| 항목 | 결정 |
|---|---|
| **코어 루프** | 걷기→입장재화(캡) → 동기화 전술 PvP / 지역 PvE / 캐릭터 성장·표현 |
| **T2W 차단** | 3겹 캡 — ① 입장권 로그캡 ② 하루 N판 ③ **걷기→전투력 0기여** |
| **P2W 차단** | 대시/보호막/이동권 판매 0. 유료=코스튬·배틀패스·광고제거·비경쟁 편의 |
| **위치 활용** | **지역 PvE**(보스/레이드/협동 방벽) — 비동기 위치 PvP 결함 구조적 회피 |
| **소셜** | 같이 걷기→입장권 2배(BLE+보행주기+경로 3중 검증) |
| **로맨스** | **캐릭터 간 내러티브 로맨스**(A안). 유저×유저 위치 로맨스 불가 |
| **남녀 분할** | **진영·내러티브 분기**(길 잇는 자 vs 길 끊는 자). 유저 성별 무관 |
| **내러티브** | α "잃어버린 길" 후보 — **작가 검증 대기, 확정 아님** |

> ⚠️ "왜 걷는가"의 스토리 당위성·캐릭터·갈등은 **작가 검증 전제**. 본 회의에서 메커니즘은 확정, 스토리는 위임.
