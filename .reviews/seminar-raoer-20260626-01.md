# 세미나: 라오어 무한매수법 → 자동매매 전략 제안

> 이사님 지시: "이 프로젝트는 알고리즘 전략이 중요. 라오어 미국주식 무한매매법을 찾아보고, 비슷한 이론/로직 세미나부터. 팀원 의견 취합해서 좋은 이론 제안."

---

## TL;DR (3줄)
1. 라오어 무한매수법은 **알고리즘으로 깔끔하게 모델링 가능** (매일 종가 + 가격 편차에 비례해 비중 조절).
2. **그러나 결정적 역설**: 라오어의 진짜 엣지는 "사람의 공포/탐욕을 이겨내는 기계적 분할매수" 자체인데, **자동매매로 기계화하면 그 행동편향 엣지가 증발** → 남는 건 순수 시장 베타 + 레짐(국면) 노출뿐.
3. 따라서 **라오어를 그대로 자동화하는 건 권하지 않음**. 제안 = "라오어 분할매수 뼈대 + **레짐(국면) 필터**(강세만 매수, 횡보·하락엔 현금 대피)" 하이브리드.

---

## 1. 라오어 이론 — 한눈에 (알고리즘 명세)

| 구성 요소 | 내용 |
|---|---|
| **자본 할당** | 전체 시드를 **40분할** |
| **매수 트리거** | **매일 종가**, LOC(Limit-On-Close) 주문 |
| **비중 함수(핵심)** | 평단가 대비 **하락할수록 더 많이, 상승할수록 적게** 매수 → 평균단가 체계적 하락 |
| **매도 룰** | "무한매수"=매도 없음 / "무한매매"=익절·리밸런싱 포함 (구분 필요) |
| **전제** | 미국 지수·성장주의 **장기 우상향** + 변동성 수확 |

**비중 함수 후보 (직원1)**
- 검증용(단순): `수량 = Base × (1 - λ·d)` — `d` = 평단가 대비 현재가 편차, `λ` = 가속도 파라미터 1개. 하락(d<0)일수록 수량 증가.
- 운영용(권장): **계단식/시그모이드** — 극단 하락 시 비용 폭증을 막는 **상한 캡핑(cap)** 내장.

**학술 위치**: DCA(비용평균법) + Value Averaging(Edleson)의 하이브리드. 단, 효과는 **전부 "장기 우상향" 전제에 의존**.

---

## 2. 한 판 시뮬레이션 (체감 — 왜 위험한가)

> 이사님 피드백 반영: 숫자로 보여주기.

**시나리오 A — 우상향장 (라오어가 빛나는 곳)**
- 시드 $10,000 / 40분할 / TQQQ(3배 레버리지) 가정
- 6개월 +30% 보고 사례 존재. 하락 때 더 사서 평단가 낮추고, 반등 시 수익 폭발. **그림 아름다움.**

**시나리오 B — 횡보/하락장 (평단가 함정) ★**
- 예: 2000–2009 S&P 500 "잃어버린 10년". 10년간 본전.
- 분할매수는 평단가를 낮추지만, **지수가 우상향하지 않으면 언덕을 깎으며 손실만 누적**.
- 자동매매는 "감정 없이 계속 삼" → 사람이라면 멈출 타이밍에도 기계는 계속 현금을 태움. **자동화가 역으로 독이 됨.**

**시나리오 C — 레버리지(TQQQ) decay**
- 3배 레버리지는 일일 복원리상 횡보장에서 **변동성 드래그**로 가치 깎임. 라오어+TQQQ 조합은 강세장에선 왕이지만, 횡보·하락에선 **자본 고갈 속도가 3배**. 비추천 근거.

---

## 3. 핵심 통찰 — "자동화의 역설" ★

> 두 직원이 독립적으로 도달한 같은 결론.

- 라오어 무한매수법이 **개인 투자자에게 통하는 이유** = 인간의 본성(공포에선 도망, 환호에 끌려매수)을 **기계적 룰로 억누르기 때문**. "행동편향 회피"가 엣지의 본체.
- 그런데 우리는 이걸 **자동매매**로 만들려 함. 자동매매는 이미 감정이 없다 → **라오어의 행동편향-회피 엣지는 자동화 순간 상실**.
- 결론: 라오어를 그대로 자동화하면 **"순수 시장 베타(미국주식 상승)에 배팅하는 것"과 동일**해짐. 알파(초과수익) 희박.

→ 이사님 의도("알고리즘 전략이 중요")를 살리려면, **라오어의 '무엇'을 가져갈지** 설계해야 함. (§5 제안)

---

## 4. 위험 3가지

1. **횡보장 평단가 함정** — 우상향이 멈추면 손실 확대. 가장 치명적.
2. **백테스트 우상향 편향** — 대부분의 "라오어 검증"이 2010–21 강세장 데이터. 이건 **전략의 우수성이 아니라 국면의 행운** (López de Prado / DSR 원칙 위반).
3. **소자본 일일매수 비용** — 40분할 일일 매수 시 수수료·스프레드 비중 과대. 자본이 작을수록 비용이 수익 잠식.

---

## 5. 제안 — 하이브리드 전략 (직원2)

> 라오어를 버리지 말고, **"뼈대"만 가져와 레짔(국면) 필터를 결합**.

**전략 골격:**
1. **라오어 분할매수 뼈대** — 단, **현물 1배** (TQQQ 배제). 비용·decay 위험 차단.
2. **+ VAA/DAA 모멘텀 레짔 필터** — 시장이 **강세 국면일 때만 매수**. 횡보·하락 국면 감지 시 **현금 대피(킬스위치)**. (이게 없으면 §2-B 함정에 빠짐.)
3. **+ 빈도 조정** — 일일 매수 → **주간/월간**으로 완화 (수수료 절감). 소자본에 필수.
4. **+ DRIP(배당재투자)** 결합 — 복리 효과.

**왜 이게 나은가:** 라오어의 약점(하락장에 계속 삼)을 레짔 필터로 보완. 자동매매의 이점(기계적 실행)은 살리되, "무조건 사는" 우매함은 제거. → 비로소 자동매매 알고리즘으로서 의미.

**주관적 적합성 등급: 중** (LOC 주문은 IBKR TWS API로 기술적 구현 가능. 단, 알파는 여전히 베타+레짔 필터 품질에 좌우.)

---

## 6. 이사님 결정 사항

이사님, 아래 중 방향을 정해주시면 ADR로 발전시킵니다.

- **(a) 라오어 단독** — 가장 단순하지만, §3 역설로 알파 희박. v1 피드백(엣지 부족)과 같은 함정.
- **(b) 하이브리드(★ 팀 추천)** — 라오어 뼈대 + 레짔 필터 + 빈도 완화. 자동매매 전략으로서 의미 있음.
- **(c) 라오어 대신 다른 방향** — 예: §5-2의 VAA/DAA 모멘텀 전략 자체, 또는 v1의 구조적 차익으로 회귀.
- **(d) 추가 조사** — 특정 이론(예: Value Averaging, Just Keep Buying) 심화 세미나 2라운드.

## 7. v1 초안과의 관계 (투명 보고)

| | v1 초안 | 라오어/하이브리드 |
|---|---|---|
| 패러다임 | **구조적 단절 차익** (방향중립) | **방향성** (미국주식 우상향 베팅) |
| 알파 원천 | 스프레드/왜곡 포착 | 시장 베타 + 레짔 타이밍 |
| 충돌 여지 | 서로 다른 전략 클래스. **한 시스템에 공존 가능** (v1 백본 + 전략 모듈). | — |

→ 충돌 아님. **v1의 "구조적 메커니즘 사고"**(왜 엣지가 남는가)를 라오어 평가에 적용한 결과가 §3 통찰. 하이브리드(b)의 레짔 필터 모듈은 v1 백본 위에 올릴 수 있음.
